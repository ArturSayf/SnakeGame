Snake Game - Змейка
Классическая игра "Змейка", реализованная на C# с использованием Windows Forms.

Описание игры
Игрок управляет змейкой, которая перемещается по игровому полю и собирает еду. С каждым съеденным кусочком змейка увеличивается в длине, а скорость игры возрастает. Игра заканчивается при столкновении змейки с границей поля или с собственным хвостом.

Особенности
Игровое поле: 15x15 клеток
Управление: клавиши WASD или стрелки

Система очков:
Еда в углах поля: +1000 очков
Еда на границах поля: +500 очков
Еда в центре поля: +250 очков
Прогрессия сложности: скорость увеличивается с каждым съеденным кусочком
Статистика: отображается длина змейки, текущая скорость, количество очков и съеденной еды

Управление
Действие	Клавиши
Вверх	↑ или W
Вниз	↓ или S
Влево	← или A
Вправо	→ или D
Начать игру	Пробел
Выход	Escape
Технические детали
Язык: C#

Фреймворк: .NET 8.0
Технология: Windows Forms

Архитектура:
Отрисовка через событие Paint с двойной буферизацией
Игровой цикл на основе Timer (интервал 300 мс, уменьшается на 5 мс за съеденную еду)
Хранение змейки: LinkedList<Point>
Обработка столкновений: проверка границ и самопересечения

Структура проекта
FrmSnakeGame.cs - основная логика игры и отрисовка
FrmSnakeGame.Designer.cs - дизайнер формы и компоненты
Program.cs - точка входа в приложение
SnakeGame.csproj - файл проекта

Системные требования
Windows OS
.NET 8.0 или выше
Разрешение экрана: не менее 800x600

Установка и запуск
Склонируйте репозиторий
Откройте решение в Visual Studio 2022
Соберите проект (Build → Build Solution)
Запустите игру (Debug → Start Without Debugging)
Или используйте dotnet CLI:

bash
dotnet build
dotnet run
